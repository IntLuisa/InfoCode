# InfoCodigo

O InfoCodigo é uma plataforma desenhada para desenvolvedores e agências que precisam organizar a fase mais crítica de um projeto: o alinhamento de expectativas. Ele centraliza o levantamento de requisitos, a geração de contratos e a coleta de dados técnicos em um só lugar.

## Features

- Laravel ^11.31
- PHP ^8.2
- Jetstream ^5.3
- Inertia ^1.0.14
- PHPUnit ^11.0.1
- Vue ^3.3.13
- Tailwind ^3.4.0
- Eslint 
- Prettier 
-Font Awesome 6.6.0
- Mysql 
- SQlite
